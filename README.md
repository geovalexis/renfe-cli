RENFE TIMETABLES CLI
====================

Usage:
```
./renfe.py -h
Usage: renfe.py [options]

Options:
  -h, --help            show this help message and exit
  -y YEAR, --year=YEAR
  -m MONTH, --month=MONTH
  -d DAY, --day=DAY
  -o ORIGIN, --origin=ORIGIN
  -t TO, --to=TO
```
**Note**: defaults to `today`, from `Sils` to `Barcelona`
